**Purpose:**
The purpose of setting up OpenVPN auto start is to ensure that connections from the server to the cloud service environment are re-established, when there are unexpected VPN disconnects (e.g., power outages, operating system reboots).

**I.** Open vpn apps chosse **Settings** tabin the left panel 
![[Pasted image 20250604162112.png|300]]

**II.** Choose option **Connect latest** and now you can test shutdown OS or restart, sure OpenVPN connected again automatic.
![[Pasted image 20250604162210.png|300]]
