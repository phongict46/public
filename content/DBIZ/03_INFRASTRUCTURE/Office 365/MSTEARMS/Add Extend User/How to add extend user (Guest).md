**Step by step configure below.**

**I.** Login page Guest account management with account have permission [link](https://admin.microsoft.com/?login_hint=phongpt%40dbiz.com&source=applauncher#/GuestUsers) chose **Add a guest user** for create new guest account.
![[Pasted image 20250610135848.png]] 

**II.** You will have two options: **Create User** and **Invite User** in this case choose **Invite User** with condition Guest already have email account (Gmail, Office 365, etc.) 
![[Pasted image 20250610140350.png]]

**III.**  Fill information necessary the same with the images below. => choose **Invite**.
![[Pasted image 20250610141740.png]]
![[Pasted image 20250610141832.png]]

**III.**  If you see notification the same with images below, you can sure Guest user already receive email invite send to inbox of them.
![[Pasted image 20250610142211.png]]

**IV.**  Notice to Guest user check inbox email and choose link invite same with the images below.
![[Pasted image 20250610142724.png]]

**V.** Check code sent to the user or use password login email individually.
![[Pasted image 20250610143039.png|300]]

**VI.** If everything is true you will see the Guest account in the list below.
![[Pasted image 20250610145702.png]]

**VII.** Now you can add users to the group chat follow guide. [[Create group chat in ms teams]]
