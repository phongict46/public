### **How to add Signature for outlook office 365 online**

![[Pasted image 20250524114203.png]]

#### **II.** <mark style="background: #FF5582A6;">1</mark> Select the settings icon to go to the user settings page => <mark style="background: #FF5582A6;">2</mark> select Account => <mark style="background: #FF5582A6;">3</mark> Signatures => <mark style="background: #FF5582A6;">4</mark> create a name for the signature => 5 fill in the signature information => <mark style="background: #FF5582A6;">6</mark>, <mark style="background: #FF5582A6;">7</mark> choose the automatic signature when sending new emails and replying to emails.
![[Pasted image 20250524120729.png]]
