##### **How to create new account office 365**

**I.** Log in account manage office 365 [link](https://admin.microsoft.com/Adminportal/Home?#/homepage)
![[Pasted image 20250524105208.png]]

**II.** Choose <mark style="background: #FF5582A6;">1</mark> **Users** => <mark style="background: #FF5582A6;">2</mark> **Active users** tab in the left panel => <mark style="background: #FF5582A6;">3</mark> **Add a user**.
![[Pasted image 20250524105614.png]]

**III.** Fill in the account information to be created, make sure to select the correct domain currently used **dbiz.com**.
![[Pasted image 20250524110303.png]]

**IV.** Issuing the necessary licenses for new accounts.
![[Pasted image 20250524110405.png]]
**V.** For users who usually select **Next** for the next step, for users who need certain special  management rights such as (creating accounts, deleting accounts, adding licenses to accounts...), configurations can be set up at the current step.
![[Pasted image 20250524110433.png]]

**VI.** Select **Finish** to proceed to the next step.
![[Pasted image 20250524110502.png]]

**VII.** In this step, you can choose 2 methods depending on the company's policy.

- <mark style="background: #FF5582A6;">Option1:</mark> **Print** the account information file and proceed to update the file to OneDrive to store the new account information and use this file to send an email to the company admin.
- <mark style="background: #FF5582A6;">Option2:</mark> Send an email directly to the company admin via email.

![[Pasted image 20250524110833.png]]
